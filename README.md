## Live Site

[epsictron.github.io](https://epsictron.github.io)

## What's Here

- **Home** (`index.html`) – Portfolio landing page
- **Resume** (`resume.html`) – Professional experience, education, projects, and publication
- **Interactive DSP Demos**
  - [Frequency Resolution & Windowing](https://epsictron.github.io/frequency-resolution.html) – DFT, spectral leakage, Hamming window
  - [Impulse Response & Convolution](https://epsictron.github.io/impulse-response.html) – LTI systems, convolution integral
  - [Pole-Zero & Z-Plane](https://epsictron.github.io/pole-zero.html) – Zero placement, unit circle, notch filters

## Tech Stack

- Vanilla HTML, CSS, JavaScript (no build tools, no frameworks)
- All signal processing computed in-browser using HTML5 Canvas
- Hosted via GitHub Pages
