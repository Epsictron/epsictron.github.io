Impulse Response & Convolution

Watch how an LTI system with impulse response h(t) = e−t responds to different input shapes. Drag C towards zero and see the output converge to the impulse response itself.

h(t) = e⁻ᵗ (reference)

Key Concepts

  • Impulse Response h(t) — for this first-order system h(t) = e⁻ᵗ·u(t). It is the output when the input is a Dirac delta δ(t).
  • Convolution y(t) = (x ∗ h)(t) — the output for any input is the convolution integral ∫ x(τ)·h(t − τ) dτ. The system "remembers" its past inputs.
  • Unit-area inputs — both inputs are normalised so ∫ x(t) dt = 1. As C → 0 the rect pulse → δ(t), so the output → h(t) = e⁻ᵗ.
  • Try this: Drag C down to 0.1 and watch y₁(t) (blue) converge to the dashed green reference h(t). This is the definition of impulse response made visual.
